Object Detection

MediaPipe Object Detection คือโซลูชันสำหรับการตรวจจับวัตถุในภาพหรือวิดีโอแบบเรียลไทม์ ที่สามารถระบุตำแหน่งและจำแนกประเภทวัตถุได้ เหมาะสำหรับงานประมวลผลภาพ

last updated October 5, 2024

Image Segmentation

Image Segmentation คือการแบ่งภาพออกเป็นส่วนต่างๆ เพื่อแยกวัตถุหรือพื้นที่ตามพิกเซล ช่วยให้เข้าใจและวิเคราะห์ภาพได้ละเอียดมากขึ้นในงานประมวลผลภาพ

last updated October 5, 2024

Interactive Segmentation

Interactive Segmentation คือการแบ่งภาพโดยที่ผู้ใช้สามารถโต้ตอบและควบคุมกระบวนการแบ่งส่วน ช่วยให้การระบุวัตถุหรือพื้นที่ในภาพมีความแม่นยำและตรงตามความต้องการของผู้ใช้มากขึ้น

last updated October 5, 2024

HandLandmarker

การตรวจจับและระบุตำแหน่งข้อต่อต่าง ๆ ของมือ เช่น นิ้วและข้อมือ ในภาพหรือวิดีโอแบบเรียลไทม์ เหมาะสำหรับการใช้งานในแอปที่ต้องการการควบคุมด้วยท่าทางมือ เช่น AR, VR

last updated October 5, 2024

Pose Detection

คือการตรวจจับและระบุตำแหน่งของข้อต่อต่าง ๆ บนร่างกายมนุษย์ เช่น หัว ไหล่ ข้อศอก และเข่า ในภาพหรือวิดีโอแบบเรียลไทม์ ใช้เพื่อวิเคราะห์ท่าทางการเคลื่อนไหวในงานต่าง ๆ เช่น ฟิตเนส การเล่นเกม หรือการแพทย์
